O Que é o projeto?

O nome TrOpE vem de um acrônimo referente a sistema para relacionamento Entre Trabalhador e Oportunidades de Emprego.

TrOpE se apresenta como um conjunto de extensões próprias para Redes Sociais e para o Joomla, além de vir a ser desenvolvido no futuro para outros CMS ou CMFs.

TrOpE como apresentado na internet, é um projeto que visa agregar pessoas interessadas em desenvolver um sistema que Relacione o Trabalhador com Oportunidades de Empregos.

TrOpE também define os requisitos necessários ao desenvolvimento do sistema de Relacionamento entre Trabalhador com Oportunidades de Empregos existentes.

A palavra Trope

  • Trope, na literatura é uma figura de linguagem ou forma de expressão que discorre sobre a criatividade das palavras.
  • Trope, na linguística é um forma retórica que consiste em brincar com as palavras.
  • Trope, na geometria é o inverso de um nó em uma dada curva ou superfície, ou uma linha tangente a uma data superfície.

Trope pode ser entendido como a Inovação dos conceitos ao relacionamento da empresa com o trabalhador e o envolvimento das políticas de trabalho.

Porque o Joomla

Bem quando o projeto foi concebido a instituição que o demandava, estava migrando seus sites para Joomla, inclusive o WebService que iria atender requisições externas de outra instituições e sites.

Claro há muitas outras opções portanto é fundamental que o os módulos sejam construídos em camadas, sendo as camadas de integração com o sistema de persistência plugavel e com o sistema de apresentação, e CMS ou outro gerenciador de aplicação plugavel aos módulos do core.

O que o Projeto Precisa?

O projeto TrOpE precisa agora de programadores autodidatas e que adotem o projeto visando torna-lo real e também que aproveitem a oportunidade para se promoverem e o projeto levando o as empresas como um reflexo de bom trabalho.

O que o Sistema TrOpE tem?

TrOpE ainda está em planejamento, mas já tem os seguintes requisitos:

  • ser desenvolvido como um componente do Joomla 1.5.21 ou superior
  • Fornecer um meio fácil e rápido para o Trabalhador se cadastrar no sistema via site.
  • Permitir que o Trabalhador Cadastre seu currículo, identificando seções, e perfil (básico, médio e avançado, personalizados) para seu currículo de forma a permitir que um currículo seja apresentado de várias formas.
  • Permitir que o Trabalhador se candidate a Uma vaga, solicitando mais informações
  • Permitir que Empregadores se cadastrem rapidamente no Site
  • Permitir que Empregadores identifiquem pessoas autorizadas a Gerenciar oportunidades
  • Permitir que Empregadores cadastre oportunidades de empregos
  • Permitir que Empregadores escreva pequenos scripts de analise de perfil do candidato a oportunidade
  • Permitir que o Empregador convoque Trabalhadores para Oportunidades

Requisitos secundários porem importantes

  • Integração a Redes Sociais na seguinte ordem:
    • Twitter -> Publicar oportunidades e candidatos com currículos autenticados
    • LinkedIn -> sincronizar perfil do LinkeIn com currículos, publicar oportunidades
    • Facebook -> Publicar oportunidades
    • Orkut -> Publicar oportunidades
  • Integração com redes sociais altamente qualificadas como:
    • Lattes
    • Redes Sociais cientificas.
